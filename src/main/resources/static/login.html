API Access

For programmatic access to the API, you will need to obtain an access token from AWS Cognito and include it in your requests as a Bearer token.

Example:

    curl -H "Authorization: Bearer YOUR_ACCESS_TOKEN" https://your-api-endpoint.com/api/resource
